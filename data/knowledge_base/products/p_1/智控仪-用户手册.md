
# 智控仪产品知识库

## 一、 产品概述与生态

### 1. 产品简介与定位
产品名称为智能增氧机控制器，也称鱼开心鱼塘测控仪或鱼开心智控仪。该产品是自主研发的新型智能设备，旨在帮助水产养殖户实现智能化养殖。产品具备高品质、高效率、高智能、安装简单、使用方便等特点。通过实现无人值守与智能诊断，有效降低翻塘风险并节约管理成本。

### 2. 研发背景
产品历时3年研发，调研了1268家鱼塘主和水产公司，拜访70余名行业专家，并聘请20余名国内外知名技术工程师组建专业研发团队。

### 3. 产品生态矩阵
围绕水产养殖和工厂化养殖，提供完整的软硬件生态系统：
- 核心硬件：水产养殖智能测控仪、鱼开心水质监测仪、工厂化循环水养殖PLC控制柜、水产水质检测传感器、监控摄像头、断电报警器、远程温度监测仪。
- 系统与解决方案：智能温控系统、智能排污系统、数字化智慧渔业养殖平台、增氧机与投料机等配套设备、工厂化模式建设整体解决方案。

## 三、 安装与接线指南

### 1. 安装前准备与安全规范
- 拆装需断电，严禁带电操作。
- 通电前检查接线端子是否压紧铜线而非线皮，可用手拉拽确认。
- 所有线头必须接触良好，否则可能烧毁设备。
- 总空开、防浪涌保护器、漏电保护器为安全保护装置，务必安装可靠。

### 2. 单相220V接线方案
- 小功率（1.5KW以下）：交流输入220V -> 总空开 -> 测控仪 -> 漏电保护器 -> 单相增氧机。需安装防浪涌保护器并接地线。在小程序或APP端请勿将电流设定超过12A。
- 大功率（1.5KW以上）：必须外接220V交流接触器。路径为：交流输入220V -> 总空开 -> 测控仪 -> 外接220V交流接触器 -> 漏电保护器 -> 单相增氧机。注意：增加接触器后，增氧机电流不经过设备，APP与小程序无法监控电流数据。

### 3. 三相380V接线方案
- 小功率（3KW以下）：交流输入ABC380 -> 防浪涌/空开 -> 测控仪 -> 漏电保护器 -> 三相增氧机。N线可不接。
- 四线大功率（3KW以上）：需外接交流接触器。输入ABCN380，供电端和接触器控制端接AN相（220V线圈）。若外接380V线圈接触器，控制器输入三相ABC，输出A和C接接触器A1和A2。
- 三线大功率（3KW以上）：输入ABC380，需外接380V线圈交流接触器。

### 4. 混接方案
- 三相380V与220V小功率混接：适用于三相3KW以下和单相1.5KW以下设备。进线必须采用三相四线制（ABCN380），经防浪涌后分路连接不同漏电保护器。
- 三相380V与220V大功率混接：适用于三相3KW以上和单相1.5KW以上设备。进线采用三相四线制，大功率设备需外接交流接触器。

### 5. 线材与接地规格
- 进线（空开进出线和控制器进线）：6平方及以上多芯纯铜线。
- 出线（控制器出线）：2.5平方及以上多芯纯铜线。
- 接地（防浪涌PE线）：地线大于10平方，地线电阻小于4欧。

## 四、 传感器安装与维护

### 1. 传感器接头与防水处理
- 标配航插头对插接头。对插后用力拧紧，用防水贴纸包好，置于浮球之上防浸入水中。接头切勿浸入水中。
- 设备右下角最大支持4路传感器（1#至4#）。使用4芯线束，卡扣朝上接入。建议单个传感器接入最左侧1#接口。安装前务必断电。

### 2. 传感器投放位置与深度
- 探头距离岸边3米以上，距离增氧机5米左右。
- 下水深度30至50厘米。
- 浮筒需固定好防止随风漂移。

### 3. 溶解氧传感器使用与校准
- 采用荧光法测量。使用时务必去掉带海绵保护套，装上保护罩。
- 切忌用锐器接触表面膜头。贮存和携带时底部加薄海绵垫保护，使用时拆除并妥善保存。
- 偏差校正：若存在较大偏差，清洗后静置空气中15分钟，对比不同温度下空气数据，按现场校正方法修正。

### 4. 日常清洗与保养
- 每10至15天清洗一次。提出水面后用干净抹布、软毛刷和清水轻轻清洗底部，或用湿润柔软布/海绵擦拭前端荧光膜。顽固污垢可用自来水加少量家用洗涤液清洗。切勿用力擦洗。
- 不使用时，将前端放入带有湿润海绵的保护套内保持湿润。若长期干燥导致测量漂移，需在水中浸泡1至7天恢复。
- 若清洗后发现荧光膜破损或手机报警探头故障，立刻联系厂家。

## 五、 设备操作与界面说明

### 1. 本地触摸屏操作
- 主界面：顶部显示服务器连接状态、时间、触摸音开关。左侧轮流展示传感器数据、电压、电流。中间为4路输出开关快速控制。右侧为设置区。
- 联动设定界面：可设置输出线路、传感联动开关、编号、定时任务。
- 操作逻辑：手动开关设备后变为手动状态，需再次点击联动或定时恢复自动。断网后本地参数保持不受影响。

### 2. SIM卡安装与网络配置
- 设备内置贴片SIM卡，非必要无需理会。若网络不稳定可自行安装。
- 支持移动、联通、电信4G网络。卡片规格为15x12mm的MicroSIM卡（中卡）。
| 卡类型 | 长(mm) | 宽(mm) | 厚(mm) |
| :--- | :--- | :--- | :--- |
| SIM卡(MiniSIM) | 25 | 15 | 0.76 |
| Micro SIM卡 | 12 | 15 | 0.76 |
| Nano SIM卡 | 12 | 9 | 0.67 |
- 必须断电安装，不支持热插拔。缺口朝外、芯片朝上。检测到双卡后自动切换网络。

## 六、 软件APP与小程序使用

### 1. 设备绑定与共享管理
- 绑定：通过扫描设备唯一序列号二维码绑定。
- 手机号共享：输入已注册APP的手机号搜索添加，可取消管理资格。
- 二维码共享：被分享人扫码申请，塘主在消息中心审核同意。

### 2. 数据监控与状态查看
- 显示传感器数据（溶氧、水温、PH、氨氮等），支持点击查看详情或报表。
- 显示设备输入电压、各路输出电流及运行状态（在线/离线）。

### 3. 控制模式与参数配置
| 模式名称 | 功能描述 | 注意事项与限制 |
| :--- | :--- | :--- |
| 手动模式 | 手动点击控制设备的开/关 | 开启时，传感联动与定时联动自动关闭 |
| 传感联动 | 通过传感器参数设定，自动联动控制设备开关 | 必须接传感器；低于下限开启，高于上限关闭；开启时手动模式关闭 |
| 定时联动 | 通过设定好的定时参数，自动控制设备开关 | 单路最大支持8组定时任务；开启时手动模式关闭 |

- 参数配置：支持自定义输出名称、功率配置。可开启电流监测（超限语音电话通知）和电流保护（超限自动关闭设备）。

### 4. 报警消息中心
- 分为报警消息、系统消息、分享消息三个标签页。
- 支持按设备、日期查询筛选。显示设备名称、序列号、报警方式、状态、时间及详情。
- 分享消息用于处理他人扫码共享设备的申请，可进行同意或拒绝操作。

## 七、 报警机制与安全防护

### 1. 报警类型与触发条件
- 断电/断网报警：立即通过语音电话通知，不受常规报警方式控制，强制进行语音报警。
- 缺相报警：监测到缺相立即语音电话通知。
- 电压报警：输入电压超过上下限阈值触发。可开启电压保护，高于上限时停止输出供电。
- 传感器报警：
  - 标准模式：溶氧低于下限或高于上限立即报警，持续低于下限每下降0.3立即报警。
  - 自定义模式：触发低于下限立即报警，后续持续低于下限按自定义分钟数间隔报警。
  - 普通模式：低于下限或高于上限立即报警。
- 电流报警：实际电流低于下限或高于上限触发。

### 2. 报警通知方式
- 支持语音电话通知。
- 默认提醒对象为塘主，可添加已分享的联系人为报警联系人，并设置拨打优先顺序。

## 八、 日常维护与注意事项

### 1. 设备养护
- 定期保养，非专业人士请勿拆开设备，擅自拆卸造成事故公司概不负责。
- 传感器参数专业性强，如有不明请联系厂家，随意设定造成损失厂家概不负责。

### 2. 环境与使用注意事项
- 避免传感器表面被阳光暴晒。
- 禁止用手直接触摸荧光膜，不可用指甲等硬物刮擦，避免施加机械应力。
- 荧光膜头部严禁使用酒精、丙酮等有机溶剂清洗。
- 注意设备防水。
- 若电网无保护装置，必须安装防浪涌保护器，防止烧坏控制器或雷击引起火灾，安装位置应远离易燃易爆物品。
- 低温场景（昼夜温差大，低温小于7度）：需断开电机或其他负载，建议保持智控仪通电状态。
- 防潮建议：环境湿度大或南风天易受潮时，建议保持通电。若断电超过3小时，建议使用吹风筒对着输入端电源线孔吹热风通风3分钟。
