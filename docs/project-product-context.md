# 产品上下文 — Auto Content Platform

## 产品类型

工具型 SaaS 管理后台（后台/仪表盘/工具）。

## 目标用户

内容运营人员，单人使用为主。

## 主要任务

1. **生成内容**：输入主题/产品/公司 → AI 生成文章
2. **管理账号**：添加平台账号 → 导入登录态 → 检测会话
3. **发布内容**：选择内容 + 选择账号 → 一键发布
4. **跟踪状态**：查看发布记录，重试失败任务

## 使用环境

- 桌面浏览器（管理员日常工作）
- Windows 本地开发环境
- 不需要移动端操作（移动端仅查看）

## 核心流程

```
生成内容 → 预览确认 → 选择账号 → 发布
                ↑                    ↓
           重新生成              ［成功/失败］
                                   ↓
                              失败可重试
```

## 约束

- 单人使用，不需要权限系统
- 本地运行，不需要部署到公网
- 文章只允许发布一次（不可重复发布）
- 对应平台的账号只能发布对应平台的内容
